geno-kaggle

Kaggle benchmarking skills for Claude Code. Create, run, monitor, and review benchmark tasks on the Kaggle platform.

Commands

CommandDescription
/gt-create-benchmark-kaggleCreate a self-contained benchmark notebook for Kaggle
/gt-kaggle-benchmarks-task-generateScaffold a new benchmark task folder structure
/gt-kaggle-benchmarks-task-reviewPull and review results from a Kaggle benchmark run
/gt-run-kaggle-benchPush, run, monitor, and debug a notebook on Kaggle
/gt-upload-kaggleUpload a notebook to Kaggle
/gt-kaggle-discussionScrape competition discussions and generate insights

Prerequisites

  • Kaggle CLI v2.0+ (pip install kaggle)
  • Kaggle API credentials at ~/.kaggle/kaggle.json

Install

$ ./install.sh

# Or via the geno-tools ecosystem installer:
$ cd ../geno-tools && ./install.sh

Ecosystem

geno-tools — orchestrator + general tools geno-research — deep research workflows geno-media — media creation (audiobooks, video) geno-kaggle — Kaggle benchmarking skills
